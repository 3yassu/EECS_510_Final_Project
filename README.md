# EECS_510_Final_Project
Parses C for correct curly braces and parentheses  
To run, ensure rustup is installed on your system. It can be installed from here: http://rustup.rs/.  
Then, clone or download and extract this repository.  
To compile, run `cargo run` in the `src` folder.     
Feel free to modify pass.c/fail.c with your own test cases!
