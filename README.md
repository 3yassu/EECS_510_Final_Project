# EECS_510_Final_Project
Parses C for correct curly braces and parentheses  
2 dependencies:
  cargo/rustup (can be installed from here: http://rustup.rs/.),
  git
Run commands
```
git clone https://github.com/3yassu/EECS_510_Final_Project.git
cd EECS_510_Final_Project/src/
cargo run
```
The code can be run in any directory within the project file but the pass/fail files are in the src directory.
Feel free to modify pass.c/fail.c with your own test cases!
